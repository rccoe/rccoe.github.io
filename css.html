Using CSS on your NeoCities Page

The goal of today is to apply what you’ve learned about CSS to your own web page. By the end of today, you should have:

  • 2 CSS rules using class selectors. For example:
  •  .mr-robert {
  •     color: black;
  •  }
  • 2 CSS rules using ID selectors. For example:
  •  #mr-erty {
  •     font: 12px;
  •  }
  • 2 CSS rules using element selectors. For example:
  • body  {
  •     font: 14px;
  •  }
  • 1 Comment above each CSS rule describing what the rule does in plain english:
  • . For example:
  • /* This CSS rule makes the font for text in the body element 14 pixels */
  • body  {
  •     font: 14px;
  •  }
